# XPP Shop 项目说明

这是一个打工赚钱小游戏，使用Python开发的GUI应用程序。

## 项目概述

XPP Shop 是一个基于 tkinter 的打工赚钱模拟游戏，玩家可以通过各种方式赚取游戏内货币，包括打零工、入职公司、学习提升学历等。游戏包含背包系统、商店购买、存档功能等元素，支持自动更新。

## 核心功能

- **打工赚钱**: 提供多种零工选择，如送外卖、快递分拣等
- **公司入职**: 基于学历要求的招聘市场，可入职不同公司获得日薪
- **学历系统**: 通过自学、补习班等方式提升学历值
- **状态系统**: 饱食度和心情值影响工作效率
- **商店系统**: 购买食物等物品
- **背包系统**: 管理已购买的物品
- **存档系统**: 自动保存游戏进度
- **自动更新**: 检查并下载最新版本

## 文件结构

- `Main.py`: 游戏主程序，包含GUI界面和核心逻辑
- `Model.py`: 游戏数据模型，定义全局变量
- `Page.py`: 游戏页面相关数据，包含公司信息
- `ArchivalProcessing.py`: 存档处理系统
- `ProGressBar.py`: 进度条界面组件
- `CheckUpdateProgram.py`: 自动更新功能
- `XPPShopEXE/`: 存放打包好的可执行文件

## 技术栈

- Python 3.13
- Tkinter (GUI界面)
- 标准库: threading, queue, random, datetime 等
- 第三方库: 无特殊依赖

## 启动方式

直接运行 `Main.py` 文件启动游戏：
```
python Main.py
```

## 游戏机制

- 钱数: 游戏内货币，用于购买物品和服务
- 学历值: 代表能力，影响可入职公司的范围
- 饱食度: 影响赚钱效率(0-100)
- 心情值: 影响赚钱效率(0-100)
- 工作单位: 入职公司后获得稳定日收入

## 开发约定

- 代码使用中文注释和变量名
- GUI界面使用固定尺寸(800x600)
- 存档文件保存在系统用户目录下
- 使用UTF-8编码处理文件
